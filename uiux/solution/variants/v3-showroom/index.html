Showroom = просторная сетка 2×2 для визуалистов. Ключевая деталь — hero-блок 146px, где реальные цвета фонов из backdrops.json работают подложкой, а поверх них стоит крупное превью 96px и видимая цепочка моделей до 3-5шт без клика. Узнавание подписки происходит за 200мс. Палитра, шрифт и примитивы — те же, что в оригинале Gift Satellite. Снижение плотности (4 вместо 6) окупается тем, что пользователю больше не нужно читать текст чтобы отличить «Ice Cream Divine Justice» от «Ice Cream Ghost Biters».

Hero-подложка

1 фон = сплошной radial-gradient (центр/край/узор из backdrops.json). 2-4 фона = диагональное/конусное деление. 5 фонов = conic с равными клиньями. 25+ = многоточечный кастом-градиент. 0 фонов = subtle checkered placeholder.

Модели

0 моделей — текст «Любые модели» в центре hero. 1-5 моделей — мини-цепочка 26px glass-chips внизу hero. >5 моделей — большое превью коллекции + бейдж «12 моделей» в углу.

Состояния

0шт — 2px красная обводка + warn-бейдж «Куплено 0/N» + зачёркнутая цена. Покупка выкл — dim 55% + off-бейдж. Нотиф выкл — bell-off + dim-маркеты + текст «Нотиф. выкл» вместо иконок.

Bulk-режим

Toolbar заменяется счётчиком «Выбрано N из 45» + Отмена/Все. 22px чекбоксы в левом верхнем углу hero. Floating action-bar над табами: Удалить / Выкл / Вкл / Дубл. Выделенные карточки — primary-border + inset glow.

# Плана работ — Redesign карточки подписки Gift Satellite

> Статус: зафиксировано, ждёт апрува концепции и ответов на 5 блокеров (см. конец файла).
> Дата: 2026-04-14.

## Концепция

**«Плотная карточка с управляемой информационной иерархией»**: каждая карточка — самодостаточная визитка подписки, где превью + цветная подложка фонов + строка маркетов делают 80% работы по распознаванию без чтения. Текст — метаданные второго плана. Главный экшн — Switch покупки в нижней трети. «Изменить/Удалить» прячутся в `⋯` меню или свайп, чтобы освободить место.

## Информационная иерархия внутри карточки

1. **Заголовок**: промпт-имя пользователя (bold, truncate 40 симв). Справа — иконка `info` (`i`).
2. **Превью** (~80–90px высоты):
   - Цветные полоски фонов как фоновая подложка.
   - Поверх — иконка коллекции (0 моделей) / цепочка до 5 моделей / общая иконка коллекции (>5).
3. **Метаданные** (микротекст, каждая строка со своей иконкой):
   - Коллекция / «Все коллекции»
   - Модели (текст)
   - Узоры (текст)
   - Числовой паттерн (текст)
4. **Строка маркетов** — до 5 цветных кружков логотипов (Portals/Fragment/MRKT/Tonnel/GetGems). Раздельно для покупки и нотификаций.
5. **Экшн-бар**: Switch покупки + цена «до X ◇» + счётчик «N шт» + `⋯` меню.

## Состояния карточки

| Состояние | Индикация |
|---|---|
| Обычное | Стандартная отрисовка |
| Покупка включена | Switch ON, цена и маркеты видны |
| Покупка выключена | Карточка dimmed −40%, бейдж «Покупка выкл» слева вверху |
| **Куплено, 0 шт** | **Красная обводка 2px + красный бейдж `!` + «Куплено 0/N»** (настойчиво) |
| Нотификации включены | Иконка `bell` + цена + маркеты |
| Нотификации выключены | Иконка `bell-off` серая, текст «Нотификации выкл» |
| 0 моделей выбрано | Текст «Любые модели», превью — иконка коллекции |
| 1 модель выбрана | Её имя + её иконка |
| 2–5 моделей | Цепочка иконок |
| >5 моделей | Общая иконка коллекции + «N моделей» |
| 0 фонов | Текст «Любые фоны», подложка прозрачная/кастом |
| 1 фон | Цвет подложки = этот фон |
| 2–X фонов | Подложка делится на X частей, каждая крашена |
| >X фонов | Кастомный градиент-подложка |
| Массовый режим | Чекбокс слева сверху, лёгкий зум |
| Выбрано в массовом режиме | Чекбокс заполнен, лёгкая primary-обводка |

Значение X — **4** (моё предложение, требует апрува).

## Два режима плотности

- **Compact (2×3)** — как сейчас, 6 на экран, узкая полоска для фонов, маркеты 14px.
- **Comfortable (2×2)** — 4 на экран, больше воздуха, цепочка моделей до 5 видна сразу, маркеты 18px.

Переключатель в верхнем toolbar (рядом с сортировкой): list / grid-compact / grid-comfy.

## Модалка полного описания

Bottom-sheet 85% высоты:
- header: промпт-имя + Info + кнопка `x`
- секции: Коллекция / Модели (скролл-чипы) / Фоны (скролл-чипы с цветами) / Узоры (чипы) / Паттерн номера
- блок Покупка: состояние / цена / количество / маркеты
- блок Нотификации: состояние / цена / маркеты
- действия: Изменить, Удалить, Switch покупки

## Режим массового выбора

- Вход: кнопка «Выбрать» в toolbar (iOS Telegram блокирует долгий тап HTML-элементов).
- Верх: счётчик «Выбрано N из 45».
- Bottom-sheet: Удалить / Выкл покупку / Вкл покупку / Дублировать / Отмена.

## Риски и допущения

| # | Риск | Митигация |
|---|---|---|
| 1 | `.tgs` не рендерится в браузере без lottie-player | Использую `assets/originals/*.png` как placeholder; lottie-web — если успею |
| 2 | Нет логотипов маркетов | SVG-заглушки: буквы P/F/M/T/G на цветных кругах |
| 3 | Нет иконки бренда Gift Satellite | Вытаскиваю из скринов или рисую минималистичный gift-силуэт |
| 4 | iOS Telegram блокирует долгий тап | Мультивыбор через кнопку в toolbar |
